# Jellyfin Enhanced

The essential enhancement suite for Jellyfin, bundling advanced features and customizations into one convenient plugin.

## Features

- **Advanced Keyboard Shortcuts** - Comprehensive hotkeys for navigation and playback control
- **Smart Bookmarks** - Save and jump to specific timestamps with visual timeline markers
- **Custom Pause Screen** - Beautiful overlay with media info when paused
- **Jellyseerr Integration** - Search, request, and discover media directly from Jellyfin
- **Quality Tags** - See 4K, HDR, Atmos at a glance on posters
- **Genre Tags** - Identify genres instantly with themed icons
- **Rating Tags** - TMDB and Rotten Tomatoes ratings on posters
- **Multi-language Support** - Available in 12+ languages

## Quick Start

1. Go to **Dashboard** → **Plugins** → **Catalog**
2. Add repository: `https://raw.githubusercontent.com/n00bcodr/jellyfin-plugins/main/10.11/manifest.json`
3. Install **Jellyfin Enhanced**
4. Restart Jellyfin server

See [Installation Guide](installation.md) for detailed instructions.
