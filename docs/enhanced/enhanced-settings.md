# User configuration of Enhanced Settings

## Enhanced Panel

Access user-configured settings via the Enhanced panel:

| Shortcuts | Settings |
|-----------|----------|
| ![Shortcuts](../images/shortcuts.png) | ![Settings](../images/settings.png) |
<!-- relative paths for images -->

**Open Panel:**
- Click **Jellyfin Enhanced** in sidebar
- Press `?` keyboard shortcut

**Tabs:**
- **Shortcuts** - Customize keyboard shortcuts
- **Settings** - Enable/disable features, adjust positions

**Settings Persistence:**
- Settings saved to browser localStorage
- Per-user configuration
- Sync across devices (same browser profile)

### Feature Toggles

Most features can be enabled/disabled individually:

1. Open Enhanced panel
2. Go to Settings tab
3. Toggle features on/off
4. Changes apply immediately (no restart needed)

**Examples of Toggleable User Features:**
- Quality Tags
- Genre Tags
- Language Tags
- Rating Tags
- People Tags
- Pause Screen
- Auto-skip Intros
- Auto Picture-in-Picture
- Review tags
- And more...
