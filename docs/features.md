# Features

## Playback & Controls

### Keyboard Shortcuts
Comprehensive hotkeys for navigation, playback control, and more. Press `?` to view all shortcuts.

### Smart Bookmarks
Save timestamps and jump to specific moments with visual timeline markers on the video player.

### Custom Pause Screen
Beautiful overlay with media info, plot, and spinning disc when you pause a video.

### Auto-Skip Intros/Outros
Seamlessly skip intros and outros automatically (requires Intro Skipper plugin).

## Discovery & Integration

### Jellyseerr Integration
Search, request, and discover media directly from Jellyfin's search interface.

**Setup:**
1. Go to plugin settings → Jellyseerr tab
2. Enter your Jellyseerr URL and API key
3. Enable search integration

### ARR Integration
Quick links to Sonarr, Radarr, and Bazarr on item detail pages (admin only).

### Streaming Provider Lookup
See where else your media is available to stream across multiple regions.

## Visual Enhancements

### Quality Tags
Display quality information (4K, HDR, Atmos) directly on posters.

### Genre Tags
Identify genres instantly with themed icons on posters.

### Language Tags
Display available audio languages as country flags on posters.

### Rating Tags
Show TMDB and Rotten Tomatoes ratings on posters and in the video player.

### People Tags
Display age and birthplace information for cast members.

## Customization

### Custom CSS Styling
Extensively customizable appearance for all tags and features.

### Custom Branding
Upload your own logos, banners, and favicon.

### Multi-language Support
Available in 12+ languages with community translations.
