Watchlist Settings
Adds a powerful, multi-tab watchlist to Jellyfin.
This feature is powered by and loaded directly from the excellent ranaldsgift/KefinTweaks project. Check it out for more great tweaks!
Enter the commit hash or branch name from the KefinTweaks repository (e.g., "master" or "1d9fe7efdae8..."). Defaults to 'master' if left empty.
This controls which version of the watchlist scripts are loaded. In most cases, you need not change this.
info

To View Your Watchlist:

To see your watchlisted items, you need to create a new view using the Custom Tabs plugin.

  1. Install the Custom Tabs plugin and all it's pre-requisites.
  2. Navigate to Dashboard > General > Custom Tabs and add a new tab.
  3. Add "Display Text" according to your preference
  4. Copy the code below and paste it into the 'HTML Content'.
<div class="sections watchlist"></div>
<style>
  .sections.watchlist {
    padding-top: 20px; /* Adjust/remove this value as needed for your theme */
  }
</style>
Note: Depending on your theme, you may need to adjust or remove the `padding-top` value in the style block to prevent the content from appearing behind the header.
